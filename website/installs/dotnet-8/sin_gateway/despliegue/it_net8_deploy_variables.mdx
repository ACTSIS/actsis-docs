# Variables de despliegue sin Gateway

Esta página contiene las variables necesarias para el despliegue de una aplicación .NET 8 sin Gateway.

## Variables de entorno

- `ASPNETCORE_ENVIRONMENT`: Define el entorno de ejecución (Development, Staging, Production)
- `DOTNET_PRINT_TELEMETRY_MESSAGE`: Controla los mensajes de telemetría
- `DOTNET_SYSTEM_GLOBALIZATION_INVARIANT`: Configuración de globalización

## Variables de aplicación

- `ConnectionStrings__DefaultConnection`: Cadena de conexión a la base de datos
- `Logging__LogLevel__Default`: Nivel de registro predeterminado
- `AllowedHosts`: Hosts permitidos para la aplicación
