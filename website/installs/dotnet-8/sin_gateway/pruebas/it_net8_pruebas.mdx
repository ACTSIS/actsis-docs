# Pruebas para instalación sin Gateway

Esta página contiene las pruebas necesarias para verificar que la instalación de .NET 8 sin Gateway funciona correctamente.

## Verificación básica

1. Comprobar que el sitio responde correctamente
2. Verificar los logs de la aplicación
3. Comprobar la conexión a la base de datos

## Pruebas de funcionalidad

- Prueba de autenticación
- Prueba de acceso a recursos protegidos
- Verificación de rendimiento básico
